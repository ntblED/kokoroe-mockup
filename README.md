# kokoroe-mockup
The demo experience for the Kokoroe ("School-Safe" AI) Project. This is, in no way, a fully polished experience, and does not contained all the planned features for the full release.

## Brief Introduction
Kokoroe ("心得", pronounced "ko-ko-roh-eh") is a project that is currently in development. It aims to be an AI assistant that's designed to guide students by asking questions, giving simple explainations, and helpful advice. It's also designed to allow students to come up with their own answers and not rely on the AI for direct answers and revealing hints.

## Running Locally
To run the project demo locally, you can follow these steps:
1. Clone the Git repository:
`git clone https://github.com/ntblED/kokoroe-mockup`
2. Install the requirements with:
`pip install -r requirements.txt`
3. Run the server with:
`python mockup/__main__.py`
